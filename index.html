Welcome to the Guess The Number Wiki

Guess The Number is a game created by Asep1582 (RaspingWheel22)

Where you guess a number between 0 and the number limit with a certain amount of Gueses. There are 2 gamemodes currently released in selected beta. But another gamemode is being released soon, making it 3. This Gamemode is called Medium Easy Gamemode.

There are multiple game modes of Guess The Number. (Gamemodes Listed below (in order))

  • Guess The Number Easy Gamemode (Instructions to download here)
  • Guess The Number Medium Easy GameMode
  • Guess The Number Medium Gamemode
  • Guess The Number Hard Mode
  • Guess The Number Extreme Hard Mode (Pending if release/developed further)
  • Guess The Number Extra Hard GameMode (The Hardest Gamemode)

Note: Guess The Number Extreme Hard Mode may not be an offical gamemode when the Full Game is released and/or gamemodes are released.
