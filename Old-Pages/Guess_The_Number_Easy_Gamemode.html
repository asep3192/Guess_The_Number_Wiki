What is Guess The Number

Guess The Number is a Game Created by Asep1582 (RaspingWheel22) Where you guess a number between 0 and the number limit with a certain amount of Gueses. There are 2 ganmemodes currently released. But another gamemode is being released soon, making it 3. This Gamemode is called Medium Easy Gamemode.

Medium Easy Mode

Medium Easy Gamemode is similar to Medium Gamemode but it slitly increases the probaility of winning and getting the number correct in 9 trys instead of 8. The number between 0 and 300 stays the same.
